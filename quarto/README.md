# Interaktive Lernressource

`umsetzung.qmd` ist die abzugebende interaktive Lernressource.
Die dokumentierten Quellbausteine der einzelnen Interaktionen liegen
unter `code/`. `references.bib` verweist auf die gemeinsame
Literaturdatenbank im Projektstamm.

Vom Projektstamm aus rendern:

```sh
quarto render quarto
```

Die fertige Lernressource liegt anschließend unter
`docs/umsetzung.html`.
